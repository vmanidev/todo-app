# Todo App

A simple & efficient todo application to manage your tasks easily. 

## Features

- Add, Edit, & Delete tasks
- Mark tasks as completed 
- Save tasks locally 
- Dark / Light mode support 

## Installation 

### Clone the repository 

```bash 
git clone https://github.com/vmanidev/todo-app.git
cd todo-app 
```

## Tech stack 

- HTML 
- CSS 
- JavaScript
